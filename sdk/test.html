🚀 How Integration Works

1
Add one <script> tag to your site with your API key. NexAuth injects the login button automatically.
2
User enters credentials. NexAuth hashes the password in the browser with SHA-256 (Web Crypto API) — raw password never leaves the device.
3
Only the hash is sent to your backend. AI threat detection runs on every request. JWT token is returned and stored.
